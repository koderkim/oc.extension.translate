Install extension: go to Extensions->Install menu and upload zip archive.
After installation go to "System->Users->User group" menu, select the desired group (as default is Administarator) and edit access&modify permission: check "extension/extension/translate" in both and save.
After saving permission go to Extensions->Extensions menu, select type as Translate. In loading list select "DeepL Translation", install it and edit/save settings.
On product editing page in admin panel will be added button/link before product title \//Translate all (as source)\\/ on all languages tab. 
When clicked, the title and description fields will be translated into other (available) languages. The selected language will be considered the input language.
